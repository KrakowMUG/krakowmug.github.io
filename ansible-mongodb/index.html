What is Ansible?


Ansible is an IT Automation tool for configuration, management, software deployment and ad hoc task execution. It is quite similar to Chef or Puppet but it's more appealing to us because it is more flexible, and written in Python.

Ansible is a Top 10 Python project on GitHub with over 5 000 stars

Ansible features


  • doesn't require agent on managed servers

  • uses SSH for communication, no custom PKI required

  • easy to use YAML based configuration files

  • ~190 out-of-the-box modules (e.g MySQL, Redis, Riak, PostgreSQL)

  • also MongoDB, but only user management
