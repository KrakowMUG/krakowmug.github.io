What is Ansible?

Ansible is an IT Automation tool for configuring management, software deployment and ad hoc task execution. It is quite similar to Chef or Puppet but it's more appealing to us because it is written in Python.

Ansible is a Top 10 Python project on GitHub with over 4,000 stars

Ansible features

  • doesn't require agent on managed servers
  • it uses ssh for communication no custom security infrastructure
  • easy to use language called Playbooks
  • ~150 modules out of the box (e.g MySQL, Redis, Riak, PostgreSQL)
  • also MongoDB, but only user management
